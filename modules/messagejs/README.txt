MessageJS
=========

This module allows messages from the message module to be displayed within a
jGrowl notification via the Node.js Notification module included in the
Node.js integration suite.

Setup
=====

1. Follow the installation instructions and install the
    Node.js Integration module.

2. Install and enable the message module.

3. Install and enable the messageJS module.

4. Set permissions for the roles you with the message notifications
    to be displayed.

5. Enable notifications per message type on the appropriate management page.
    (http://example.com/admin/structure/messages/manage/<message type>)
    The setting will be in a fieldset at the bottom of the page titled
    "NodeJS Notifications"

6. Insure that your node server is configured and running. 

7. Relish in the Node-ey goodness.

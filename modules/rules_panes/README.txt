Rules Panes allows embedding Rules components as content panes in Panels (and
other modules using the CTools content panes). The parameters for the Rules
components can either entered manually by end-users in a form, or set by CTools
contexts.

A quick video showing how this works can be found here:
http://youtu.be/WAoAia5T5Ug

This project is looking for co-maintainers. If you find it cool and useful, feel
free to head over to http://drupal.org/node/1786070 and contact the maintainer.
